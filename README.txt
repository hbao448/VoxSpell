To run the spelling aid, first extract the zip file into your current directory
Then open the terminal and change to the current directory

Next type "/usr/lib/jvm/jdk1.8.0_91/bin/java -jar VoxSpell.jar" to start the spelling aid
Note that java 1.8 is required to start the program and that the resources folder must be in the same directory as VoxSpell.jar when it is executed
